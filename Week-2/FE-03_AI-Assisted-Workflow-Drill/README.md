# FE-03 – AI-Assisted Workflow Drill

## Overview

This assignment demonstrates the difference between using a vague AI prompt and a structured AI engineering workflow.

The same feature was developed using two independent prompting approaches and then compared based on correctness, accessibility, testing, edge cases, and review effort.

---

## Objectives

- Compare vague and detailed prompting
- Practice AI planning before coding
- Verify AI-generated code
- Document the workflow
- Create reusable AI development rules
