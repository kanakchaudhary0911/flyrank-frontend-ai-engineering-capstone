# Round 2 – Structured Prompt

## Objective

Build the same feature using a professional AI-assisted development workflow.

---

## Prompt

Project Stack

- React
- Vite
- Tailwind CSS

Task

Create a responsive Settings Form.

Requirements

- Use controlled form inputs.
- Validate Full Name, Email, Password, and Confirm Password.
- Display clear validation messages.
- Disable the submit button until the form is valid.
- Follow existing project structure.
- Keep the implementation modular and readable.

Accessibility

- Every input must have a visible label.
- Support keyboard navigation.
- Display accessible validation feedback.

Verification

- Write tests for the form.
- Run the tests.
- Fix any failing tests.
- Explain the files modified after implementation.

Workflow

Explore → Plan → Implement → Test → Review
