# Week 2 – Frontend AI Engineering

This folder contains all work completed during **Week 2** of the FlyRank Frontend AI Engineering program.

The focus of this week was learning how to work effectively with AI coding assistants by comparing different prompting approaches, documenting the development workflow, and establishing reusable project rules.

---

## Assignment

### FE-03 – AI-Assisted Workflow Drill

**Objective**

Build the same feature twice using two different AI prompting strategies to compare the impact of prompt quality on code quality, correctness, accessibility, testing, and review effort.

**Key Activities**

- Build a feature using a vague AI prompt
- Build the same feature using a detailed AI workflow
- Compare both implementations
- Document the complete workflow
- Create reusable AI development rules

---

## Learning Outcomes

By completing this week's assignment, I learned how to:

- Write effective prompts for AI coding assistants
- Separate planning from implementation
- Verify AI-generated code through testing
- Review AI-generated output critically
- Document an AI-assisted development workflow
- Create reusable project rules for future development
